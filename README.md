## Tech Stack

- React + DeveloperFolio template
- Sass for styling
- GH Pages for hosting (`npm run deploy`)

## Local Development

```bash
git clone https://github.com/khayem487/portfolio.git
cd portfolio
npm install
npm start
```

## Deploy

```bash
npm run deploy
```

This builds the project and publishes the `/build` directory to the `gh-pages` branch.
